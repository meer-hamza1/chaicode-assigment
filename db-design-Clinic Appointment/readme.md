Clinic Management System – ER Diagram
  Overview

This project represents the design of a Clinic Management System using an Entity-Relationship (ER) Diagram.
The system is designed to digitally manage clinic operations such as:

Doctors
Patients
Appointments
Consultations
Diagnostic Tests
Reports
Payments

The goal is to create a clean, scalable, and normalized database design suitable for a modern clinic.

  Objectives
Organize clinic operations digitally
Maintain relationships between patients, doctors, and visits
Track diagnostic tests and reports
Manage appointments and payments efficiently
Ensure scalability for future expansion
